# لغة شمس (Shms)

لغة برمجة عربية مصممة للبساطة والإبداع.

## الكلمات المفتاحية:
- `عرف [اسم] كـ [قيمة]` : لتعريف المتغيرات.
- `اطبع [نص]` : لإظهار المخرجات.
- `كرر [عدد] مرات { ... }` : لتنفيذ حلقة تكرار.
- `اذا [شرط] { ... } والا { ... }` : للجمل الشرطية.

## مثال:
عرف س كـ 10
كرر 5 مرات {
  اطبع "مرحباً يا شمس"
}
